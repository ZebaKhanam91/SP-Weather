This directory contains the code obtained from external sources that is used for the implementation of the project.
Three sources are included:
1) A. Vedaldi and A. Zisserman, Recognition of object categories practical, University of Oxford, 2015. [Online]. Available: http://www.robots.ox.ac.uk/~vgg/practicals/category-recognition/index.html. [Accessed 14 June 2017].
Files obtained from Source 1):
- computeFeatures.m
- displayRankedImageList.m
- displayRelevantVisualWords.m
- encodeImage.m
- getImageSet.m
- removeSpatialInformation.m
- setup.m
- standardizeImage.m
- trainEncoder.m
- trainLinearSVM.m

2) G. Kalliatakis, S. Ehsan, M. Fasli, A. Leonardis, J. Gall and K. McDonald-Maier.Detection of Human Rights Violations in Images: Can Convolutional Neural Networks help? Computer Vision, Imaging and Computer Graphics Theory and Applications, (VISAPP) Conference, 2017
Files obtained from Source 2):
- matdeeprep.m (with little modifications for its use in new environment)

3) Y. Altman. cprintf - display formatted colored text in the Command Window MathWorks, Inc., 2015. [Online]. Available: https://uk.mathworks.com/matlabcentral/fileexchange/24093-cprintf-display-formatted-colored-text-in-the-command-window?focused=3880898&tab=function [Accessed June 2017].
Files obtained from Source 3):
- cprintf.m
